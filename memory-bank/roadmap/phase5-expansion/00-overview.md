# Phase 5: Platform Expansion

## Overview

This phase expands the platform's reach beyond video to include the major social networks, fulfilling the promise of "MediaBlade" as a comprehensive tool.

## Goals

1.  **Text & Image Support**: Connect to platforms that primarily use text/images.
2.  **Business Integration**: Connect to LinkedIn Pages and Facebook Pages.
3.  **Real-time Social**: Connect to X (Twitter).

## Roadmap

### 1. LinkedIn Integration

- **File**: `01-linkedin.md`
- **Features**:
  - LinkedIn OAuth (OpenID Connect).
  - Profile & Company Page support.
  - Text, Image, Video posting API.

### 2. X (Twitter) Integration

- **File**: `02-twitter-x.md`
- **Features**:
  - X OAuth 2.0.
  - Tweet posting (Text, Media).
  - Thread support (optional for MVP).

### 3. Instagram & Facebook

- **File**: `03-meta-suite.md`
- **Features**:
  - Meta Graph API integration.
  - Instagram Business Account publishing (Image/Video).
  - Facebook Page publishing.

## Success Criteria

- [ ] User can post a text update to LinkedIn.
- [ ] User can tweet an image to X.
- [ ] User can post a photo to Instagram.
